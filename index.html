FASTA to sequential PHYLIP sequence converter

Try Now Get Going

A sequence alignment file in a FASTA/Pearson format, generated by multiple sequence alignment programs such as MAFFT is usually used as an input for Phylogenetic analysis tools such as RAxML in a PHYLIP format. This text alignment tool gathers information from a FASTA alignment file, and generates an output in a relaxed (sequential/non-interleaved) PHYLIP format.
