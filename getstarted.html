How to prepare an input:

  • Open your alignment file in a text editor, for example: Notepad++ or TextWrangler
  • Go to Replace and mark ‘Extended’ for search mode (in Notepad++) or ‘Grep’ (in TextWrangler)
  • Type /r/n in a 'Find what' field and replace all with space
  • Select, copy all, and paste it in the input form

Requirements

  • Input must be in a FASTA/Pearson format
  • Characters in a set of DNA/Protein sequences must be in capital letters
  • Dash symbols "-" in labels/titles must be replaced with underscores _ before submitting an input
  • Line breaks (/r/n) must be replaced with spaces before submitting an input
